# Vic-Rooming_House-assessor
Web based Rooming House interactive map assessment

## Running the App

### GitHub Codespaces (Recommended)
1. Click the green **Code** button on the repository page.
2. Select the **Codespaces** tab and click **Create codespace on main**.
3. Once the dev container starts and packages are installed, the app launches automatically and opens in your **browser** on port `8501`.

### Running Locally
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the Streamlit app:
   ```bash
   streamlit run app.py
   ```
3. Open `http://localhost:8501` in your browser.
